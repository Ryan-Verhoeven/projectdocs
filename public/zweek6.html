Interactie en Knoppen

Mijn ontwerpproces

In week 6 ben ik gestart met het omzetten van mijn Figma-ontwerp naar HTML en CSS in Visual Studio Code. In plaats van me enkel te richten op het visuele aspect, heb ik me vooral beziggehouden met het functioneel maken van knoppen en navigatie. Het correct laten werken van deze interacties vroeg extra aandacht en zorgde ervoor dat de structuur van de code duidelijk en logisch moest zijn.

Door mijn beperkte voorkennis in VSC heb ik niet alles verder geoptimaliseerd, maar de basisfunctionaliteit van het ontwerp werkt correct en sluit aan bij het uitgewerkte high-fidelity ontwerp in Figma.

Verschillende Knop Stijlen

Primary Button

De hoofdknop voor de belangrijkste actie op een pagina. Trekt de meeste aandacht.

Secondary Button

Secundaire knoppen voor minder belangrijke acties. Vaak met een outline stijl.

Success Button

Groene knoppen voor positieve acties zoals opslaan of bevestigen.

Danger Button

Rode knoppen voor gevaarlijke acties zoals verwijderen.

Rounded Button

Knoppen met volledig ronde hoeken voor een moderne look.

Code Voorbeelden

HTML Button Structuur

<button class="primary-button" onclick="mijnFunctie()">
  Klik mij!
</button>

<button class="secondary-button">
  Meer info
</button>

CSS Button Styling

.primary-button {
  background: linear-gradient(135deg, #667eea, #764ba2);
  color: white;
  padding: 0.8rem 1.5rem;
  border: none;
  border-radius: 8px;
  cursor: pointer;
  transition: all 0.3s ease;
}

.primary-button:hover {
  transform: translateY(-3px);
  box-shadow: 0 5px 15px rgba(102, 126, 234, 0.4);
}

JavaScript Click Event

function mijnFunctie() {
  alert('Knop geklikt!');
}

let counter = 0;

function increment() {
  counter++;
  document.getElementById('counter').textContent = counter;
}

function decrement() {
  counter--;
  document.getElementById('counter').textContent = counter;
}
