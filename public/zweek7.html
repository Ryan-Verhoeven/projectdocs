CSS Styling en Layout

Ontwerp proces

In week 7 lag de focus voornaamlijk op de styling en lay-out van mijn code. Ik heb gewerkt aan het verfijnen van de visuele opbouw zodat het geheel minder standaard oogt. Door aanpassingen in CSS heb ik geprobeerd meer structuur te brengen in de interface. Het doel was om de code niet alleen functioneel te maken, maar ook visueel overtuigend.

Belangrijke CSS Properties

Kleuren

color background-color

Bepaal de tekst- en achtergrondkleuren van elementen. Gebruik hex codes, RGB of kleur namen.

Lettertypen

font-family font-size font-weight

Stijl je tekst met verschillende lettertypen, groottes en gewichten.

Spacing

margin padding

Creëer ruimte rondom en binnen elementen voor een betere lay-out.

Borders

border border-radius

Voeg randen toe aan elementen en maak afgeronde hoeken.

Flexbox

display: flex justify-content align-items

Maak flexibele lay-outs die automatisch aanpassen aan verschillende schermformaten.

Grid

display: grid grid-template-columns

Creëer complexe twee-dimensionale lay-outs met rijen en kolommen.

Positionering

position top left

Bepaal de exacte positie van elementen op de pagina.

Transitions

transition transform

Voeg vloeiende animaties toe aan je elementen voor een betere gebruikerservaring.

Code Voorbeelden

Basis CSS Styling

.mijn-element {
  color: #667eea;
  background-color: white;
  padding: 20px;
  margin: 10px;
  border-radius: 10px;
  box-shadow: 0 5px 15px rgba(0, 0, 0, 0.1);
}

Flexbox Voorbeeld

.flex-container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 1rem;
  flex-wrap: wrap;
}

.flex-item {
  flex: 1;
  min-width: 200px;
}

Grid Layout Voorbeeld

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 2rem;
}

@media (max-width: 768px) {
  .grid-container {
    grid-template-columns: 1fr;
  }
}

Hover Effecten en Transitions

.button {
  background: #667eea;
  color: white;
  padding: 1rem 2rem;
  border-radius: 50px;
  transition: all 0.3s ease;
}

.button:hover {
  transform: translateY(-3px);
  box-shadow: 0 10px 25px rgba(102, 126, 234, 0.4);
}
